So... I want to create Android app. And it should be unique :)

(It will be probably very hard to get recycle bins positions from Prazske sluzby a.s. so so far I will only incorporate looking for nearest scrap yards, but it should not be hard to change that if the data will be available)

For next week I want to get some ideas on graphical layout and basic functionality.
